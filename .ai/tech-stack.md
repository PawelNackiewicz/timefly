# TimeFly Technology Stack

## Frontend

- **Astro** - Modern static site builder with React integration
- **React** - JavaScript library for building user interfaces
- **Tailwind CSS** - Utility-first CSS framework for rapid UI development
- **shadcn/ui** - High-quality, accessible UI components built on Radix UI and Tailwind

## Backend & Database

- **Supabase** - Open source Firebase alternative providing:
  - Authentication and user management
  - PostgreSQL database
  - Storage
  - Real-time subscriptions
- **PostgreSQL** - Advanced open-source relational database (managed by Supabase)

## Testing

- **Playwright** - End-to-end testing framework (to be implemented post-MVP)

## Deployment

- **Vercel** - Platform for frontend deployment, optimized for Astro

## Development Considerations

- MVP approach prioritizes core functionality with room for scalability
- Initial focus on web app before potential mobile development
- Security handled through Supabase authentication and Astro best practices
